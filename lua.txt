Lua (/ˈluːə/ LOO-ə; from Portuguese: lua [ˈlu(w)ɐ] meaning moon) is a lightweight, high-level, multi-paradigm programming language designed mainly for embedded use in applications.[2] Lua is cross-platform software, since the interpreter of compiled bytecode is written in ANSI C,[3] and Lua has a relatively simple C application programming interface (API) to embed it into applications.[4]
